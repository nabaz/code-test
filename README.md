# code-test

## To Run Django App Go to ./backend and run python install -r requirements.txt
## Create .env file in the root folder and set these values:
```DB_NAME= 'dbname'
DB_USER='dbuser'
DB_PASSWORD='dbpassword'
DB_HOST='localhost'
DB_PORT='5432'```

## then run migration 
```python manage.py migrate```
